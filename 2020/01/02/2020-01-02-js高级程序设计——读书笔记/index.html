js高级程序设计——读书笔记

发表于 2020-01-02 | 分类于 js

STEP01–基础总结

在HTML中使用JavaScript

<script> 标签

在HTML页面中插入javascript的主要方法就是使用<script>标签。

<script>标签有以下主要属性：

  1. src：（可选）用于链接外部文件
  2. async：（可选）表示异步执行此代码，仅支持外部文件。
  3. defer：（可选）表示延迟执行此代码，仅支持外部文件。

对于后两种属性的区别，一图胜万言：

注: （上图来源与网络）在图中，绿色（parser分析）表示html加载，蓝色（net网络）表示网络下载，红色（execution执行）表示执行代码

如果没有defer/async，那么<script>标签则会按序加载执行后才会开始加载<script>标签后的内容，所有<scirpt>标签的位置一般会直接放到<body>标签中页面内容的后面，这样加载html文件时，就不会导致浏览器在呈现页面时出现延迟。<script>标签是按序执行的，当很多个<script>标签时应需注意前后位置是否会影响页面的正常加载。

js代码有两种方式引入，一种是通过src属性进行对js外部文件的引入，另一种则是直接在<script>标签内写js代码，需要注意一点，如果在一个script标签中，既设有src属性，又在标签中写js代码，那么只会执行外部文件而不会去执行标签内部的js代码。

<noscript>标签

如果浏览器不支持脚本或者浏览器禁用了js，那么就会显示<noscript>标签内的内容，如果浏览器启用js则不会显示<noscript>标签内的内容。

基本概念

语法

区分大小写

javascript是区分大小写的

标识符

标识符：即为一个属性、函数、变量的名字，或者是函数的参数。

标识符命名规则：

  1. 必须是字母、下划线（_）、dollar符号（$）开头
  2. 其他字符可以是字母、下划线（_）、dollar（$）、数字

驼峰命名法：

是将从第二个字符串开始，将首字母大写后拼接在一起组成。例如：theFirstTry、myDear

注释

在javascript中注释方法又两种，一种单行注释（//），另一种为多行注释（/* */）

例如：

//这是一条单行注释

/*

这是一条多行注释

这是一条多行注释

*/

当然注释的内容不会在页面中显示出来，注释有利于对代码的理解。

数据类型

数据类型总共又六种，包括五种基本数据类型（Undefined,Null,String,Number,Boolean）以及一种复杂数据类型（Object）

typeof操作符

typeof操作符可以查看一个变量的类型。

注意一点，typeof操作符一般不会报错，如果去查看一个未定义的标识符的类型，则会返回undefined。

数据类型详解

Undefined类型

Undefined类型只有一个值undefined，表示只定义未赋值。

标识符在定义后，如果没有显式的赋值那么编译器会默认赋值undefined，如上var age;操作实际上等效与var age = undefined;。

Null类型

Null类型和Undefined类型一样，只有一个值为null，表示空的对象指针。

如果要定义一个对象，初始化最好显式的赋值为null。

String类型

String类型表示字符串，一般由双引号或者单引号表示。

Number类型

Number类型表示数字，包括整数，浮点数，也可以是科学计数法表示的数。(数值表示范围为-Infinity到Infinity，其中Infinity表示为无穷）

NaN：（not a number）表示非数字，任何非0的数除以0值为NaN。（console.log(typeof NaN);输出的值为Number类型）

NaN无法使用等号运算符进行判断，只能使用下列的方法进行判断：

1.isNaN():用于判断是否是一个数字，如果不是则返回false，如果是则返回true

2.Number.isNaN():用于判断值是否是NaN，如果是NaN则返回true，如果不是则返回false

数值转换：

Boolean类型：true->1,false->0

Null类型：null->0

Undefined类型：undefined->NaN

String类型：空字符串对应0,其他从字符串的第一个字符开始，一直到非数字截至，如果有数字，那么返回数字，如果没有，则返回NaN，如果数字以0开头则忽略0，如果以0x开头则转换为相对应的十进制返回（返回数字均为十进制）

转换方式如下：

1.parseInt()：返回整数

2.parseFloat()：返回浮点数

3.Number()（不常用）

Boolean类型

布尔类型的值有两个，true和false，表示真假，一般用于条件语句的判定，在条件语句中会将参数默认转换为boolean类型的值。

Boolean()：转型函数，可以将其他类型的值转换成boolean类型。

其他类型转换成false的值有：””,null,undefined,0,NaN

除了以上五种之外，其余的所有转换为boolean类型都为true
